화이트보드 가시성 보정 시뮬레이터

실제 재질 실험값을 입력하고, 그 값을 바탕으로 이상적인 표면 조건을 예측하는 간단한 모델 보정 앱입니다.

현재 표의 숫자는 모두 임시 예시값입니다. 실제 실험값이 나오면 표의 숫자를 직접 수정하십시오. 이 앱은 정답을 내는 도구가 아니라, 실험값을 기반으로 “가시성 좋은 표면의 방향”을 추정하는 도구입니다.

1. 실험 데이터 입력

모든 값은 0~100 기준입니다. 값이 클수록 해당 성질이 강하다는 뜻입니다. 예를 들어 정반사 값이 높으면 눈부심이 강하고, 지움성 값이 높으면 잘 지워지는 표면입니다.

점수 계산식

가시성 점수 = 0.45 × 글씨 대비 + 0.30 × (100 - 정반사) + 0.20 × 난반사 + 0.05 × 주관 가시성
종합 점수 = 0.70 × 가시성 점수 + 0.30 × 지움성 점수

4. 거칠기-마찰 조합별 종합 점수 지도

가로축은 거칠기, 세로축은 마찰입니다. 밝을수록 종합 점수가 높은 조합입니다.

5. 데이터 내보내기 / 가져오기

실험값을 JSON 형태로 보관할 수 있습니다. 보고서 작성 시 실험 데이터 기록용으로도 사용할 수 있습니다.
